# Викторина

Интерактивное консольное приложение для проведения тестов с подсчетом баллов и таблицей лидеров.

## Требования

- Python 3.8+
- SQLite (встроенный, дополнительная установка не требуется)

## Установка и запуск

1. Скачайте все файлы проекта в одну папку
2. Откройте терминал в этой папке
3. Запустите: `python main.py`

## База данных

Проект использует SQLite. База данных создается автоматически при первом запуске.

### Диаграмма базы данных

**Таблица `questions`** — хранит вопросы и варианты ответов:
- `id` — первичный ключ
- `text` — текст вопроса
- `option_1`…`option_4` — варианты ответов
- `correct_idx` — номер правильного варианта (1-4)

**Таблица `leaderboard`** — хранит результаты игроков:
- `id` — первичный ключ
- `player_name` — имя игрока
- `score` — процент правильных ответов
- `date_played` — дата и время игры

Связи между таблицами отсутствуют.

## Функционал

- Случайный порядок вопросов
- Подсчет правильных ответов
- Вывод процента успеха
- Сохранение результатов в таблицу лидеров
- Просмотр топ-10 лучших результатов
